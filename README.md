# [LG U+ 유레카] 미니 프로젝트 - 휴대폰 판매 시뮬레이션 시스템

## 프로젝트 요구사항
  - JDBC 와 Swing 을 이용한 휴대폰 판매관리 시스템 구현  
  - 추가 조건 자유
  - MySQL 사용

## 기술 스택

| 기술          | 사용 목적                                   |
|--------------|--------------------------------------------|
| **Java (Swing)**  | GUI 기반 시뮬레이션 View 및 로직 개발    |
| **MySQL**        | 제품 및 판매 데이터 관리                |
| **JDBC**         | 데이터베이스 연동                        |
| **Git / GitHub** | 버전 관리 및 협업                        |
